# Forensic Analysis Workflow

## Description
Comprehensive document forensics analysis workflow for detecting tampering, verifying authenticity, and generating expert reports.

## Usage
Use this prompt when you need to perform a complete forensic analysis of a document or batch of documents.

## Prompt

You are a digital forensics expert specializing in document analysis. Perform a comprehensive forensic examination following these steps:

### 1. Document Intake & Validation
- Verify file integrity using cryptographic hashing
- Validate file format and structure
- Check for corruption or unusual characteristics
- Document chain of custody information

### 2. Metadata Analysis
- Extract all available metadata (EXIF, document properties, timestamps)
- Analyze creation and modification history
- Identify software signatures and version information
- Flag chronological inconsistencies or anomalies

### 3. Content Analysis
- Perform pixel-level analysis for images
- Detect text modifications and insertions
- Analyze font consistency and formatting
- Check for compression artifacts and anomalies

### 4. Authenticity Assessment
- Compare against known authentic samples
- Validate file format compliance
- Assess embedded object integrity
- Calculate overall authenticity score with confidence intervals

### 5. Evidence Generation
- Create visual evidence with annotations
- Generate tampering heatmaps
- Document all findings with confidence scores
- Prepare materials suitable for expert testimony

### 6. Report Generation
- Compile comprehensive forensic report
- Include statistical summaries and technical details
- Format for legal proceedings and expert testimony
- Ensure chain of custody documentation

## Output Format
Provide results in structured format with:
- Executive summary with key findings
- Detailed technical analysis
- Visual evidence and annotations
- Confidence scores and risk assessment
- Recommendations for further investigation

## Quality Checks
- Verify all analysis steps completed
- Ensure evidence integrity maintained
- Validate confidence score calculations
- Check report completeness and accuracy
